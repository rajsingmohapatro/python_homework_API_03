## Dependencies
To run the project you need to have below packages
 * pandas
 * matpotlib 
 * requests
 * json
 * dotenv
 * alpaca_trade_api
 * Setup an env file which which should contain two entries
   ALPACA_API_KEY and ALPACA_SECRET_KEY with corresponding values
